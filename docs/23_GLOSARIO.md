# 23 — Glosario

**Proyecto:** AtlasHabita

| Término | Definición |
|---|---|
| API | Interfaz que permite al frontend solicitar datos al backend. |
| Capa | Representación visual o lógica de un conjunto de datos en el mapa. |
| Coroplético | Mapa que colorea polígonos según un valor. |
| CRS | Sistema de referencia de coordenadas. |
| Data quality gate | Control de calidad que permite o bloquea una publicación. |
| Dataset | Conjunto de datos con estructura y propósito definido. |
| ETL | Extracción, transformación y carga de datos. |
| ELT | Extracción, carga y transformación, normalmente en almacén destino. |
| GeoJSON | Formato JSON para datos geográficos. |
| GeoSPARQL | Vocabulario y funciones para representar y consultar datos geoespaciales en RDF. |
| Grafo RDF | Conjunto de tripletas sujeto-predicado-objeto. |
| Indicador | Medida que describe un territorio, por ejemplo alquiler o renta. |
| Knowledge Graph | Grafo de conocimiento con entidades, relaciones y semántica explícita. |
| Named graph | Subgrafo RDF identificado por una URI. |
| Ontología | Definición formal de clases, propiedades y relaciones de un dominio. |
| Perfil de decisión | Configuración de pesos y filtros para un tipo de usuario. |
| POI | Punto de interés, como hospital, estación o supermercado. |
| RDF | Resource Description Framework, modelo de datos basado en tripletas. |
| RDFS | Vocabulario para describir clases y propiedades RDF. |
| Recomendación explicable | Resultado que muestra motivos, factores y fuentes. |
| Score | Puntuación calculada para un territorio y perfil. |
| SHACL | Lenguaje de validación de grafos RDF. |
| SIG | Sistema de Información Geográfica. |
| SPARQL | Lenguaje de consulta para grafos RDF. |
| Tesela | Fragmento de mapa usado para carga eficiente. |
| URI | Identificador uniforme de recurso. |
| Validación geoespacial | Comprobación de geometrías, CRS y relaciones espaciales. |
| WGS84 | Sistema de coordenadas usado habitualmente en web maps. |
